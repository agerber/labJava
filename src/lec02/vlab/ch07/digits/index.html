Detecting Accounting Fraud
Understand the Problem
  • Given: A large data set such as population or gross domestic product by country
  • Look at distribution of digits
  • Shouldn't each digit be equally probable?
  • The first digit is not equally distributed (Benford's Law)
  • Task: Print frequencies of 1, 2, 3, ..., 9 as first digit
  • Numeric data from various sources from the web
  • Web pages pasted into text files
  • Numbers are mixed with text
  • Numbers can have commas (16,249.95)
  • Convert any sequence of 0-9 . , to a number
  • May fail; for example 130.65.86.66
  • Ignore numbers < 1
Design an Algorithm 
Ask the user for the file name
Construct a scanner for reading from the file
Read all numbers from the scanner
Get the digit distribution for the numbers
Print the distribution
